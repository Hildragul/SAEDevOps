Configuration du système et outils d’administration

L'objectif est de découvrir plus en détail la configuration du système, les accès utilisateurs, la supervision des ressources, l'explorateur de fichiers, etc.

Configuration Système (msconfig)

L’outil msconfig est conçu pour analyser les problèmes du système au démarrage. Pour le lancer, rechercher msconfig dans la barre d’outils.

Il contient 5 onglets :

  1. General : permet de choisir quels appareils et services lancer au démarrage
  2. Boot : définir les paramètres de lancement
  3. Services : liste tous les services Windows actifs ou non
  4. Startup : autrefois utilisé pour gérer les éléments au démarrage, désormais déplacé dans le gestionnaire des tâches
  5. Tools : accès à divers outils Windows avec description

Gestion informatique

Outil Windows regroupant plusieurs composants :

  • System Tools
    • Planificateur de tâches : planifie des tâches à une date/heure précise
    • Observateur d’événements : retrace l’activité de l’ordinateur
    • Dossiers partagés : affiche les dossiers partagés entre utilisateurs
    • Performance : affiche les performances d’un système ou d’une application
    • Gestionnaire de périphériques : gère les périphériques physiques (activer/désactiver)
  • Storage
    • Windows Server Backup : permet de faire une sauvegarde d’un serveur Windows
    • Gestion des disques : ajouter, partitionner, renommer des disques durs
  • Services and Applications : permet de configurer les services et l’accès à distance

Information Système

Affiche une vue complète du matériel, des composants système et de l’environnement logiciel, utile pour diagnostiquer des problèmes informatiques.

  • Ressources physiques : infos sur la gestion matérielle
  • Composants : liste les composants physiques et logiciels
  • Environnement système : infos sur les logiciels intégrés et installés

Moniteur de ressources

Affiche l’utilisation du CPU, de la mémoire, du disque et du réseau par processus. Fournit aussi les détails sur les processus liés aux fichiers et modules.

Command Prompt (CMD)

Interpréteur de commandes système. Il permet de gérer des fichiers, processus, réseaux, etc.

Quelques commandes utiles :

  • hostname : affiche le nom de la machine
  • whoami : affiche l'utilisateur actuel
  • ipconfig : affiche les paramètres réseau
  • /? : affiche l’aide d’une commande

Éditeur de registre (regedit)

Permet de modifier la base de registre de Windows. Elle contient les paramètres du système, des applications et des pilotes.

Les registres stockent notamment :

  • Profils utilisateur
  • Applications installées et leurs types de documents
  • Paramètres d’affichage (icônes, dossiers)
  • Matériel connecté au système
  • Ports utilisés

Conclusion

La room couvre les outils accessibles via msconfig, mais aussi ceux qu'on peut lancer directement via le menu Démarrer. Des raccourcis et commandes sont également partagés pour faciliter leur exécution sans passer par les menus traditionnels.
