Structure de notre page web

Pour notre SAE, nous avons conçu une page web structurée de manière claire et responsive grâce à l'utilisation de Bootstrap. Voici les principales parties qui composent notre page :

1. L'en-tête dynamique

La balise <header> accueille un en-tête généré dynamiquement par JavaScript, partagé entre plusieurs pages pour une meilleure maintenabilité. 

<header>
        <!-- Le header sera inséré dynamiquement ici -->
    </header>

2. Le corps principal

Le contenu est structuré à l’aide d’un conteneur Bootstrap pour assurer une mise en page propre et fluide. On y trouve des titres, paragraphes et listes pour organiser le texte. 

<section class="py-5">
        <div class="container px-5 my-5">
            ...
        </div>
    </section>

3. Le pied de page

Comme l’en-tête, le pied de page est inséré dynamiquement avec JavaScript. Il est situé en dehors du <body>, selon ta demande. 

<footer>
        <!-- Le footer sera inséré dynamiquement ici -->
    </footer>

4. Les scripts

En bas de page, nous avons inclus les scripts nécessaires pour le fonctionnement des composants Bootstrap et notre logique personnalisée. 

<script src="../js/scripts.js"></script>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js"></script>

5. Structure globale

Les pages complémentaires sont situées sur la page d'accueil, et chaque présentation de room est accessible via un menu de navigation.
