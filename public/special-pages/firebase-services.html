Services Firebase utilisés dans notre projet

Dans le cadre de notre SAE, nous avons fait appel à certains services Firebase pour simplifier la gestion de notre projet web. Firebase est une plateforme proposée par Google, particulièrement utile pour le développement et l’hébergement d’applications web. Voici les services que nous avons exploités :

Firebase Hosting

Firebase Hosting nous a permis de publier notre site web de manière rapide et sécurisée. Grâce à son infrastructure CDN mondiale, le chargement des pages est optimisé. Nous avons utilisé ce service pour :

  • Déployer notre site avec une simple commande depuis la ligne de commande,
  • Bénéficier automatiquement d’un certificat HTTPS,
  • Mettre à jour facilement notre application à chaque modification,
  • Prévisualiser nos modifications avant publication finale.

Firebase Cloud Storage

Pour gérer l’hébergement de fichiers, notamment des images ou documents liés au projet, nous avons utilisé Firebase Cloud Storage. Ce service nous offre :

  • Une capacité de stockage évolutive,
  • Des URLs accessibles pour partager les fichiers hébergés,
  • Un contrôle précis des accès (public ou privé),
  • Une bonne intégration avec les autres services Firebase.

Conclusion

En combinant Firebase Hosting et Cloud Storage, nous avons pu déployer une application fonctionnelle rapidement, tout en assurant la gestion efficace de nos ressources. Ces outils ont grandement simplifié la partie infrastructure du projet.
