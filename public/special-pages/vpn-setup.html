Mise en place d'une connexion VPN sur Linux pour utiliser RDP avec TryHackMe

Étape 1 : Téléchargement du fichier de configuration OpenVPN

Connectez-vous sur TryHackMe

Accédez à la section "Access" depuis le tableau de bord (l'adresse : https://tryhackme.com/access)

Téléchargez le fichier de configuration OpenVPN (.ovpn) correspondant à votre région


Étape 2 : Installation d'OpenVPN

sudo apt install openvpn

Étape 3 : Connexion au VPN TryHackMe

Placez-vous dans le répertoire où se trouve le fichier .ovpn téléchargé

Exécutez la commande suivante :

sudo openvpn votre_fichier.ovpn

Laissez OpenVPN tourner en arrière-plan tant que vous utilisez THM

Étape 4 : Vérification de la connexion VPN

Après avoir lancé OpenVPN, vous devriez voir une sortie indiquant "Initialization Sequence Completed". Vérifiez votre adresse IP pour vous assurer que vous êtes bien connecté au VPN THM :

Vérifiez votre adresse IP pour vous assurer que vous êtes bien connecté au VPN THM :

ip a | grep tun0

Si une interface tun0 est affichée, la connexion est établie.

Étape 5 : Installation d'un client RDP

Pour accéder à une machine via RDP, vous avez besoin d'un client RDP. On va utiliser remmina

sudo apt install remmina

Étape 6 : Connexion à une machine THM en RDP

Ouvrez remmina

Ajoutez une nouvelle connexion RDP

Saisissez l'adresse IP de la machine THM attribuée

Indiquez les identifiants fournis par THM

Enregistrez et connectez-vous

Conclusion

Vous êtes maintenant connecté au VPN de TryHackMe et pouvez utiliser RDP pour accéder aux machines. Pensez à fermer votre session OpenVPN lorsque vous avez terminé avec :

sudo killall openvpn
